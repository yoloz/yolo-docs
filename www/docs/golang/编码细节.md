## java 中 byte 转换

java byte 的范围是 `-128-127`，而 golang byte 是 uint8 的别名，范围是`0-255`。

```go
func javaByteToGoByte(r rune) byte {
	if r < -128 || r > 127 {
		panic("超出byte 范围")
	}
	if r < 0 {
		return byte(256 + r)
	}
	return byte(r)
}
```

:::note
实际使用无需如此麻烦，直接`b := byte(n)`即可
:::

### 字符串与数字
